# Memory Game

A fun and interactive **memory tile-matching game** built with **Kotlin**, **Spring Boot**, and **H2 Database**. Test your memory, flip tiles to find matching pairs, and compete on the leaderboard!

---

## Features

- Classic memory match game with a customizable board (choose rows × columns) (2 x 3), (4 x 4), (4 x 5)  
- Flip tiles to find matching pairs  
- Track the number of guesses per game  
- Submit your username to save your score on a persistent leaderboard  

---

## How to Run Locally

### Prerequisites

- Java 21 
- Gradle
- Modern web browser  

### Steps

1. Clone the repository:

```bash
git clone https://github.com/jclaassen123/Multilang-Portfolio/new/main/Kotlin/web
```

2. Use Gradle to start the app. Ensure you're in the same location as the cloned repo.
```bash
./gradlew bootRun
```

3. Navigate to http://localhost:8080
4.  Start Playing!
   - Choose a board size
   - Flip tiles to match pairs
   - Submit your username to record your score on the leaderboard
