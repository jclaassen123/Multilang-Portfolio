How to Play

Use the arrow keys or the W, A, S, and D keys to move every tile on the board at once.

When two tiles with the same number collide, they merge into one tile with their total value.

After each move, a new tile appears. Reach the 2048 tile to win, and keep going for a higher score if you want.

If the board fills up and no merges are possible, the game ends.
